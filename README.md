# Taylor McKendrick's Portfolio

Welcome to my personal portfolio website, built with Next.js and modern web technologies. This site serves as a showcase of my professional experience, projects, and technical skills.

## 🚀 Features

- Modern, responsive design
- Server-side rendered pages for optimal performance
- Interactive project showcase
- Professional resume access
- Contact information and social links

## 🛠️ Technology Stack

- **Framework**: [Next.js 14](https://nextjs.org)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Font**: Geist (Vercel's new font family)

## 🔧 Local Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) to view the site.
